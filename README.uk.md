# AI Supply Chain Trust

[English](README.md) | Українська

Безкоштовний сканер із відкритим кодом для оцінювання довіри до репозиторіїв і безпеки ланцюга постачання ПЗ. Публічні репозиторії GitHub оцінюються лише за актуальними доказами, які можна простежити.

Система розрізняє «ризик не знайдено» та «доказ недоступний». LLM не може створювати докази безпеки або безпосередньо змінювати оцінку.

Встановлення, API, розгортання та безпеку описано в [англійській документації](README.md).
