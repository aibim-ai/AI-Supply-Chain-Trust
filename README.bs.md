# AI Supply Chain Trust

[English](README.md) | Bosanski

Besplatan skener otvorenog koda za povjerenje u repozitorije i sigurnost lanca nabavke softvera. Javne GitHub repozitorije procjenjuje isključivo na osnovu aktuelnih i provjerljivih dokaza.

Sistem razlikuje „rizik nije pronađen“ od „dokaz nije dostupan“. LLM ne može stvarati sigurnosne dokaze niti direktno mijenjati rezultat.

Instalacija, API, postavljanje i sigurnost opisani su u [engleskoj dokumentaciji](README.md).
