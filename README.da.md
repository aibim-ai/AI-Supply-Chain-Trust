# AI Supply Chain Trust

[English](README.md) | Dansk

En gratis open source-scanner til repository-tillid og sikkerhed i softwareforsyningskæden. Offentlige GitHub-repositories vurderes kun med aktuelle og sporbare beviser.

Systemet skelner mellem “ingen risiko fundet” og “bevis ikke tilgængeligt”. En LLM kan ikke oprette sikkerhedsbeviser eller ændre scoren direkte.

Se den [engelske dokumentation](README.md) for installation, API, drift og sikkerhed.
