# AI Supply Chain Trust

[English](README.md) | Norsk

En gratis skanner med åpen kildekode for repository-tillit og sikkerhet i programvareforsyningskjeden. Offentlige GitHub-repositorier vurderes bare med oppdatert og sporbar dokumentasjon.

Systemet skiller mellom «ingen risiko funnet» og «dokumentasjon utilgjengelig». En LLM kan ikke opprette sikkerhetsbevis eller endre poengsummen direkte.

Se [engelsk dokumentasjon](README.md) for installasjon, API, utrulling og sikkerhet.
