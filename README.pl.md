# AI Supply Chain Trust

[English](README.md) | Polski

Bezpłatny skaner open source do oceny zaufania repozytoriów i bezpieczeństwa łańcucha dostaw oprogramowania. Publiczne repozytoria GitHub są oceniane wyłącznie na podstawie aktualnych i identyfikowalnych dowodów.

System odróżnia „nie znaleziono ryzyka” od „dowód niedostępny”. LLM nie może tworzyć dowodów bezpieczeństwa ani bezpośrednio zmieniać wyniku.

Instalację, API, wdrożenie i zasady bezpieczeństwa opisuje [dokumentacja angielska](README.md).
