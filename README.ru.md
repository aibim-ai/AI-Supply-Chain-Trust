# AI Supply Chain Trust

[English](README.md) | Русский

Бесплатный сканер с открытым исходным кодом для оценки доверия к репозиториям и безопасности цепочки поставок ПО. Публичные репозитории GitHub оцениваются только по актуальным и проверяемым свидетельствам.

Система различает «риск не обнаружен» и «свидетельство недоступно». LLM не может создавать свидетельства безопасности или напрямую менять оценку.

Установка, API, развёртывание и безопасность описаны в [английской документации](README.md).
