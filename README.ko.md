# AI Supply Chain Trust

[English](README.md) | 한국어

공개 GitHub 저장소를 실시간 추적 가능한 증거만으로 평가하는 무료 오픈 소스 저장소 신뢰 및 소프트웨어 공급망 보안 스캐너입니다. 신뢰 점수, 보안 컨텍스트와 취약점 검토 단서를 제공합니다.

위험이 발견되지 않은 경우와 증거를 사용할 수 없는 경우를 명확히 구분합니다. LLM은 보안 증거를 만들거나 점수를 직접 변경할 수 없습니다.

설치, API, 배포 및 보안에 대한 전체 설명은 [영문 문서](README.md)를 참조하세요.
