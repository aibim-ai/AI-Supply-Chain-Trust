# AI Supply Chain Trust

[English](README.md) | Português (Brasil)

Scanner gratuito e de código aberto para confiança em repositórios e segurança da cadeia de suprimentos de software. Avalia repositórios públicos do GitHub somente com evidências atuais e rastreáveis.

O sistema diferencia “nenhum risco encontrado” de “evidência indisponível”. Um LLM não pode criar evidências de segurança nem alterar diretamente a pontuação.

Consulte a [documentação em inglês](README.md) para instalação, API, implantação e segurança.
