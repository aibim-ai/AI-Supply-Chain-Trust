# AI Supply Chain Trust

[English](README.md) | العربية

ماسح مجاني ومفتوح المصدر لقياس موثوقية المستودعات وأمن سلسلة توريد البرمجيات. يقيّم مستودعات GitHub العامة بالاعتماد حصراً على أدلة حديثة وقابلة للتتبع.

يميّز النظام بين «لم يُعثر على خطر» و«الدليل غير متاح». لا يستطيع نموذج اللغة إنشاء دليل أمني أو تغيير النتيجة مباشرة.

راجع [الوثائق الإنجليزية](README.md) للتثبيت وواجهة API والنشر والأمان.
