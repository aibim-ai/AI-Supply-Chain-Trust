# AI Supply Chain Trust

[English](README.md) | Español

Escáner gratuito y de código abierto para la confianza de repositorios y la seguridad de la cadena de suministro de software. Evalúa repositorios públicos de GitHub únicamente con evidencias actuales y trazables.

El sistema distingue entre «no se encontró riesgo» y «evidencia no disponible». Un LLM no puede crear evidencia de seguridad ni modificar directamente la puntuación.

Consulta la [documentación en inglés](README.md) para instalación, API, despliegue y seguridad.
