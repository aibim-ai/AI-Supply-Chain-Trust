# Raw research extract — OpenSSF Scorecard

Captured: 2026-07-16
Sources: https://openssf.org/scorecard/ and https://github.com/ossf/scorecard

- OpenSSF describes Scorecard as automated security scoring for open-source projects to help assess trust, risk, and security posture.
- Tool assigns 0–10 scores to documented repository security checks.
- Interfaces include CLI, GitHub Action, JSON, public viewer, API/public datasets, and BigQuery data.
- Apache-2.0 and community governed.
- GitHub API snapshot: created 2020-10-09; active 2026-07-15; 5,583 stars; 681 forks; 427 open issues.
