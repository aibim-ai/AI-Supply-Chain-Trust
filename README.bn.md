# AI Supply Chain Trust

[English](README.md) | বাংলা

রিপোজিটরির বিশ্বাসযোগ্যতা ও সফটওয়্যার সাপ্লাই-চেইন নিরাপত্তার জন্য বিনামূল্যের ওপেন-সোর্স স্ক্যানার। এটি শুধু বর্তমান ও যাচাইযোগ্য প্রমাণ ব্যবহার করে পাবলিক GitHub রিপোজিটরি মূল্যায়ন করে।

সিস্টেম “ঝুঁকি পাওয়া যায়নি” এবং “প্রমাণ পাওয়া যায়নি”—এই দুই অবস্থাকে আলাদা রাখে। LLM নিরাপত্তা প্রমাণ তৈরি বা সরাসরি স্কোর পরিবর্তন করতে পারে না।

ইনস্টলেশন, API, ডিপ্লয়মেন্ট ও নিরাপত্তার বিস্তারিত [ইংরেজি নথিতে](README.md) দেখুন।
