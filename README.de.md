# AI Supply Chain Trust

[English](README.md) | Deutsch

Ein kostenloser, quelloffener Scanner für Repository-Vertrauen und Software-Lieferkettensicherheit. Öffentliche GitHub-Repositories werden ausschließlich anhand aktueller, nachvollziehbarer Belege bewertet.

Das System unterscheidet klar zwischen „kein Risiko gefunden“ und „Beleg nicht verfügbar“. Ein LLM kann weder Sicherheitsbelege erzeugen noch Bewertungen direkt verändern.

Installation, API, Bereitstellung und Sicherheitsdetails stehen in der [englischen Dokumentation](README.md).
