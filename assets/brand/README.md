# AI Supply Chain Trust — Brand Assets

## Concept

The **Provenance Channel** mark represents live evidence moving through source and build stages into a verified artifact. The white channel is intentionally continuous and open at both ends: trust is traceable, not assumed. The green terminal is the evaluated output.

## Core palette

- Provenance black: `#0B1117`
- Evidence green: `#16B978`
- Interface white: `#F7FAFC`

## Usage

- Use `ai-supply-chain-trust-logo.svg` on light backgrounds.
- Use `ai-supply-chain-trust-logo-dark.svg` on dark backgrounds.
- Use the standalone mark for avatars, favicons, CLI output, and compact headers.
- Use the mono mark where only one ink/color is available.
- Keep clear space around the mark equal to at least half the width of its green terminal module.
- Do not rotate the mark, recolor individual stages, close the white channel, or add shadows and outlines.

## Research archive

The `exploration/` directory contains the multi-model candidates, prompts, evaluation matrix, and the controlled synthesis used to reach the final mark.
