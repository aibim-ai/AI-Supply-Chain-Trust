# AI Supply Chain Trust

[English](README.md) | Ελληνικά

Δωρεάν σαρωτής ανοικτού κώδικα για την αξιοπιστία αποθετηρίων και την ασφάλεια της εφοδιαστικής αλυσίδας λογισμικού. Αξιολογεί δημόσια αποθετήρια GitHub μόνο με επίκαιρα και ανιχνεύσιμα τεκμήρια.

Το σύστημα διακρίνει το «δεν βρέθηκε κίνδυνος» από το «το τεκμήριο δεν είναι διαθέσιμο». Ένα LLM δεν μπορεί να δημιουργήσει τεκμήρια ασφαλείας ή να αλλάξει απευθείας τη βαθμολογία.

Για εγκατάσταση, API, ανάπτυξη και ασφάλεια δείτε την [αγγλική τεκμηρίωση](README.md).
