# AI Supply Chain Trust

[English](README.md) | Français

Analyseur gratuit et open source de confiance des dépôts et de sécurité de la chaîne d’approvisionnement logicielle. Il évalue les dépôts GitHub publics uniquement à partir de preuves actuelles et traçables.

Le système distingue « aucun risque trouvé » de « preuve indisponible ». Un LLM ne peut ni créer de preuve de sécurité ni modifier directement le score.

Consultez la [documentation anglaise](README.md) pour l’installation, l’API, le déploiement et la sécurité.
