# AI Supply Chain Trust

[English](README.md) | Italiano

Scanner gratuito e open source per l’affidabilità dei repository e la sicurezza della supply chain software. Valuta repository GitHub pubblici usando esclusivamente prove aggiornate e tracciabili.

Il sistema distingue tra «nessun rischio rilevato» e «prova non disponibile». Un LLM non può creare prove di sicurezza né modificare direttamente il punteggio.

Per installazione, API, distribuzione e sicurezza consulta la [documentazione inglese](README.md).
