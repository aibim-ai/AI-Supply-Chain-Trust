# AI Supply Chain Trust

[English](README.md) | Türkçe

Herkese açık GitHub depolarını yalnızca güncel ve izlenebilir kanıtlarla değerlendiren ücretsiz, açık kaynaklı depo güveni ve yazılım tedarik zinciri güvenliği tarayıcısıdır. Güven skoru, güvenlik bağlamı ve zafiyet inceleme ipuçları üretir.

Sistem “risk bulunmadı” ile “kanıt kullanılamıyor” durumlarını açıkça ayırır. LLM güvenlik kanıtı oluşturamaz ve puanı doğrudan değiştiremez.

Kurulum, API, dağıtım ve güvenlik ayrıntıları için [İngilizce belgeye](README.md) bakın.
