เครื่องมือตรวจสอบโอเพนซอร์สฟรีสำหรับความน่าเชื่อถือของ repository และความปลอดภัยของห่วงโซ่อุปทานซอฟต์แวร์ ประเมิน GitHub repository สาธารณะด้วยหลักฐานที่เป็นปัจจุบันและตรวจสอบย้อนกลับได้เท่านั้น

ดูวิธีติดตั้ง API การ deploy และความปลอดภัยใน[เอกสารภาษาอังกฤษ](README.md)
